# Realtor.ca Listing Filter

Live at **https://anasabushaikha.github.io/realtor/**

Paste a realtor.ca map/list search link to filter listings by parking (yes/no), title/ownership type (Freehold, Condominium/Strata, etc.), and keyword. Backed by a small Cloudflare Worker proxy (realtor.ca blocks direct browser calls to its API) — see the source project for local dev setup and the worker code.
